## 🔗 Demonstração

Acesse o projeto rodando [aqui](https://fm-blog-nextjs.netlify.app/)

## ⚙️ Instalação dos pacotes essenciais.

Instalar os pacotes:

```
pnpm install
```

Iniciar o Front-End:

```
pnpm run dev
```

## 🚀 Tecnologias

- Tailwind CSS.
